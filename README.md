---

## 🚀 Features

-   **User Management:** Secure authentication system, social login (Socialite), registration, and profile management.
-   **File Sharing:** Simple upload and download functionality with unique sharing links generated by [Sqids](https://sqids.org).
-   **Security:** Protect your shared files with optional password-protected access.
-   **Discovery:** Built-in search functionality and filtering by latest uploads or popular downloads.
-   **Admin Dashboard:** Comprehensive control panel to manage users, files, and dynamic pages.
-   **Dynamic Pages:** Integrated CMS capabilities for admins to create and manage custom content pages using [TinyMCE](https://www.tiny.cloud/).
-   **Modern UI:** Clean and responsive design using [Bootstrap 5](https://getbootstrap.com) and Sass.
-   **File Management:** Automatic file size formatting and download counters.

## 🛠️ Tech Stack

-   **Backend:** [Laravel 12](https://laravel.com), PHP 8.2+
-   **Frontend:** Bootstrap 5, Sass, Vite
-   **Database:** MySQL / PostgreSQL / SQLite
-   **Key Libraries:**
    -   `laravel/socialite` - Social authentication.
    -   `sqids/sqids` - For short, unique file identifiers.
    -   `spatie/laravel-sluggable` - For SEO-friendly page URLs.
    -   `tinymce` - Rich text editor for page management.

## 📦 Installation

To get started with Drop2Share, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/dianudi/drop2share.git
    cd drop2share
    ```

2.  **Run the automated setup script:**
    This script will install dependencies, generate the app key, and run migrations.
    ```bash
    composer run setup
    ```

3.  **Configure Environment:**
    Edit the `.env` file to set your database credentials, mail server, and Socialite keys.

4.  **Run Development Environment:**
    ```bash
    composer run dev
    ```
    *This will start the PHP server, queue listener, and Vite in parallel.*
